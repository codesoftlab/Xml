Detailed Description

Klasa XmlVect dziedzieczy po XmlObject. Zawiera tablice obiektow XmlObject_string. Posiada wszytkie metody potrzebne do parsowania do XML i odczytu. Zapis ma forme: <element> <SIZE>...</SIZE> <Tab[0].element>Tab[0].name</Tab[0].element> <Tab[n].element>Tab[n].name</Tab[n].element> </element>
